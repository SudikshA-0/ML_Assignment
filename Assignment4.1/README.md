# ANN for Digit Classification using MNIST

## Objective
Build and train an Artificial Neural Network (ANN) to classify handwritten digits using the MNIST dataset.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Model Architecture
- Input Layer (Flatten)
- Hidden Layer (128 neurons, ReLU)
- Hidden Layer (64 neurons, ReLU)
- Output Layer (10 neurons, Softmax)

## Results
- Model trained on the MNIST dataset
- Evaluated using Test Accuracy and Test Loss
- Predicted the classes of 10 test images successfully
